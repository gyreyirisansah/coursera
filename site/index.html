The central point of the system is Google Firebase real-time database. This is where the energy meter, mobile app and the web application send and retrieve data. A real-time database is one that saves data very rapidly into the database and retrieves data from it. Data is stored in JSON format and continually synchronized to each associated client. Firebase is Google provided API to create a database and fetch from it in real-time with only a few lines of code (Khedkar and Thube, 2017). The firebase database is a cloud-based database and does not need SQL-based queries to store or retrieve data. Therefore, the database is highly reliable such that data is kept even if the connection is lost (Khedkar and Thube, 2017).
